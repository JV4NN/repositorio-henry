Estructuras Condicionales Simples



No todos los problemas pueden resolverse empleando estructuras secuenciales.
Cuando hay que tomar una decisión aparecen las estructuras condicionales


En nuestra vida se nos presentan situasiones donde debemos decidir.

"Cierto o Falso..."


¿Elijo la carrera A o la carrera B?
¿Me pongo este pantalón?
¿Entro a sitio A o sitio B?
Para ir al trabajo,¿elijo el camino A o el camino B?
Al cursar la carrera, ¿elijo el turno mañana, tarde o noche?


Por supuesto que en un problema se combinan estructuras secuenciales y condicionales.
Cuando se presenta la eleccion tenemos la opción de realizar una actividad o no realizarla.
Es una estructura CONDICIONAL SIMPLE cuando por el camino del verdadero hay actividades y por el camino del falso no hay actividades.
Por el camino del verdadero pueden existir varias operaciones, entradas y salidas,
inclusive ya veremos que puede haber otras estructuras condicionales.

Ejemplo: Realizar la carga de una nota de un alumno.
Mostrar un mensaje que aprobo si tiene una nota mayor o igual a 4:

if (...){...}

Aparece la instruccion if en el lenguje JavaScript.
La condición se verifica verdadera se ejecuta todas las instrucciones que se encuentran encerradas entre las llavez de apertura y cerrado seguidas al if


Para disponer condiciones en un if podemos utilizar algunos de los siguientes operadores relacionales.


  • < mayor
  • >= mayor o igual
  • - menor
  • <= menor o igual
  • != distinto
  • == igual

Siempre debemos tener en cuenta que en la condición del if deben de intervenir una variable un operador relacional y otra variable o valor fijo.


Otra cosa que hemos incorporado es el operador "+" para cadenas de caracteres
