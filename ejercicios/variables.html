VARIABLES

Una variable es un deposito donde hay un valor.
Consta de un nombre y pertenece a un tipo.

Las variables son nombres que ponemos a los lugares donde almacenamos la información.
En JavaScript deben comenzar por una letra o un subrrayado (_),puede haber además digitos
entre los demas caracteres.
Una variable no puede tener el mismo nombre de una palabra clave del lenguaje.

Tipos de variables:

  1. Variables logicas o booleanas (boolean).
  2. Variables enteras (integer).
  3. Variables Decimales (float).
  4. Variables de cadenas (string).
  5. Variables Matrices (arrays).
  6. Variables objetos (object).
  7. Variables recursos (resource).
  8. Variables nulo (null)

Una variable puede almacenar:

  • Valores enteros(100,200,etc.)
  • Variables reales(1.24,2.90,etc.)
  • Variables cadenas de caracteres("Juan","Compras","Listado",etc.)
  • Variables logicos(true,false)
  • Existen otros tipos de variables que vermeos mas adelante.

Sintaxis:

Una variable se define anteponiendole la palabra clave "var":

var dia;

Se pueden declara varias variables en una misma linea:

var dia, mes, anio;

A una variables se le puede definir e inmediatamente inicializarla con un valor:

var edad=20;

o en su defecto en dos pasos:

var edad;

edad=20;


Para la elección de los nombres de una variable debemos elegir nombres representativos.

En el ejemplo anterior los nombres de las variables dia, mes y anio son lo suficientemente claros para darnos
una idea acabada sobre su contenido, una mala elección de nombres hubiera sido llamada a, b y c.
