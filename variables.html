VARIABLES

Una variable es un deposito donde hay un valor. Consta de un nombre y pertenece a un tipo.

Tipos de variables:

  • Variables logicas o booleanas (boolean).
  • Variables enteras (integer).
  • Variables Decimales (float).
  • Variables de cadenas (string).
  • Variables Matrices (arrays).
  • Variables objetos (object).
  • Variables recursos (resource).
  • Variables nulo (null)
